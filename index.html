Introduction to CSS

CSS- Cascading style Sheets- a programming language used to enhance the look/presentation of a website hence building a nice looking user interface and therefore improving user experience.

CSS helps handle colores, positions and sizes of a page content.

CSS Syntax - How do we write css

        
            selector{
                property: value;
                property: value;
                property: value;
            }
        
        
        
Example

        
            h1{
                color: darkgreen;
                text-decoration: underline;
            }

Where do we put our CSS

External CSS - put your code in .css file then link to the file from your html

Internal CSS- we put the styles in a style element

Inline CSS-- for styling individual elements using the style attribute
